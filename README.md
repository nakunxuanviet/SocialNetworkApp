# SocialNetworkApp
## Description:
A social networking application, the idea is almost similar to Instagram/Twister

## Technologies:
- .NET core 5.0
- CQRS, Domain Driven Design (DDD)
- SignalR
- EF Core 5
- Dapper
- IdentityServer
- SQL Server 2019
- ReactJS (Typescript)

## Hot features:
- API Versioning
- Logging with Serilog and Seq
- API Validation using Fluent Validation
